What is ScoutsLab?

ScoutsLab is an online platform for scouts and clubs with a report generator to generate game reports and anaylsis.

Scouts can
  • Write detailed anaylsis about the game, conditions, formations and set plays.
  • Share the report to the clubs and public.
  • Support the report document by uploading the images that is captured during the game.
Clubs can
  • Request report for the upgaming games.
  • Hire multiple scouts for a single report.
